# Criterion Action Plan Template (Copy/Paste)

Use this structure in prompts or post-processing so every criterion returns developmental guidance, including when already secure.

```text
Action Plan:
1) Threshold check: State exactly what evidence must be visible to meet the criterion wording.
2) Immediate next step: Give one concrete action the learner can complete now.
3) Pathway to improve: Give one stretch action that raises quality/depth beyond threshold.
```

Example sentence starter:

```text
Threshold check: Ensure evidence explicitly maps to "<criterion wording>". Next step: Add a clearer example and justify why it meets the command verb. Pathway to improve: Strengthen analysis/evaluation depth so quality is secure at assessor review.
```
